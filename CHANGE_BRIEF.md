Repo-Ready Change Brief: OpsDesk Request Form
Work Item: OD-4 (Standardized Request Form implementation) from the Sprint 1 Jira Board.

Purpose: To replace the default React "spinning logo" boilerplate with a functional, validated intake form that allows users to submit internal support requests.

Expected Behavior: * The user can enter a Title, Description, Category, and Priority.

The "Submit" button must be blocked if the Title or Description fields are empty.

Upon valid submission, the form should clear its inputs and display a "Request submitted" confirmation.
